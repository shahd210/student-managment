## 📌 Overview
This project is a **Student Management System** implemented in **C++** using Object-Oriented Programming (OOP) concepts.

It demonstrates how to manage student data such as:
- Name  
- Age  
- GPA  
- ID  
- Courses  

---

## 🎯 Features
- Create and manage student objects  
- Add and store courses using `set` (no duplicates)  
- Display student information  
- Demonstrates inheritance and polymorphism  

---

## 🧠 OOP Concepts Used
- ✔ Inheritance  
- ✔ Encapsulation  
- ✔ Abstraction  
- ✔ Polymorphism
